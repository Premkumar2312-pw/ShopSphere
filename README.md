#  ShopSphere

> A console-based E-Commerce Management System built using **Java**, **JDBC**, **MySQL**, and **Maven**.

ShopSphere is a backend-focused project designed to demonstrate how Java applications interact with relational databases using JDBC. The project follows a structured architecture and implements core e-commerce functionalities while emphasizing clean coding practices and database management.

---

##  Project Overview

The primary objective of ShopSphere is to gain hands-on experience with Java backend development by building a real-world console-based e-commerce application from scratch.

The project covers database connectivity, CRUD operations, authentication, transactions, batch processing, and layered application design using the DAO pattern.

##  Getting Started

### Prerequisites

Make sure the following software is installed:

- Java JDK 17 or later
- Maven
- MySQL Server
- MySQL Workbench
- Git
- VS Code (or any Java IDE)

---

##  Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Premkumar2312-pw/ShopSphere.git
```

### 2. Navigate to the Project

```bash
cd ShopSphere
```

### 3. Configure MySQL

- Create a database.
- Import the SQL schema.
- Update your database credentials in the configuration file.

### 4. Build the Project

```bash
mvn clean install
```

### 5. Run the Application

```bash
mvn exec:java
```

*(or run the Main class directly from your IDE)*
